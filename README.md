## ğŸ“ How It Works

```
Movie Metadata â†’ Feature Engineering â†’ Bag-of-Words â†’ Cosine Similarity â†’ Top-N Recommendations
```

1. **Feature Engineering** â€” Each movie's genres, keywords, top-3 cast, director, and overview are combined into a `tags` string.
2. **Vectorization** â€” `CountVectorizer` (5000 max features, stop-words removed) converts tags into sparse vectors.
3. **Cosine Similarity** â€” Pairwise similarity is computed across all 5000 movies.
4. **Ranking** â€” For a query movie, top-N most similar movies are returned sorted by similarity score.

## ğŸ§ª Use the Recommender Programmatically

```python
from src.recommender import MovieRecommender

rec = MovieRecommender()
rec.fit("data/tmdb_5000_movies.csv", "data/tmdb_5000_credits.csv")

# Get recommendations
recommendations = rec.recommend("The Dark Knight", n=5)
print(recommendations[["title", "similarity_score", "vote_average"]])

# Search for a title
matches = rec.search("spider")
print(matches)  # ['Spider-Man', 'The Amazing Spider-Man', ...]

# Save for later
rec.save("artifacts")

# Load later
rec2 = MovieRecommender()
rec2.load("artifacts")
```
